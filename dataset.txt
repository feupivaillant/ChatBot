Question : Qu'est-ce que Python ?
Réponse : Python est un langage de programmation interprété, polyvalent et facile à apprendre. Il est largement utilisé dans le développement de logiciels, le traitement des données, l'intelligence artificielle et bien d'autres domaines.

Question : Comment déclarer une variable en Python ?
Réponse : Pour déclarer une variable en Python, vous pouvez simplement utiliser un nom de variable suivi d'un signe égal et de la valeur que vous souhaitez lui attribuer. Par exemple : "x = 5".

Question : Comment afficher du texte à l'écran en Python ?
Réponse : Pour afficher du texte à l'écran en Python, vous pouvez utiliser la fonction "print()". Par exemple : "print('Bonjour, monde !')".

Question : Quelle est la différence entre une liste et un tuple en Python ?
Réponse : Une liste est un type de données mutable, ce qui signifie que vous pouvez modifier ses éléments une fois qu'elle est créée. En revanche, un tuple est immuable, ce qui signifie que vous ne pouvez pas modifier ses éléments après sa création.

Question : Comment importer un module en Python ?
Réponse : Pour importer un module en Python, vous pouvez utiliser l'instruction "import". Par exemple : "import math" pour importer le module mathématique.
